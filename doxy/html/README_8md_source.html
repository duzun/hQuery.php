2 ==========
3 
4 An extremely fast and efficient web scraper that parses megabytes of HTML in a blink of an eye.
5 
6 [API Documentation](https://duzun.github.io/hQuery.php/docs/class-hQuery.html)
7 
8 # Features
9 
10  - Very fast parsing and lookup
11  - Parses broken HTML
12  - jQuery-like style of DOM traversal
13  - Low memory usage
14  - Can handle big HTML documents (I have tested up to 20Mb, but the limit is the amount of RAM you have)
15  - Doesn't require cURL to be installed
16  - Automatically handles redirects (301, 302, 303)
17  - Caches response for multiple processing tasks
18  - PHP 5.3+
19  - No dependencies
20 
21 # Install
22 
23 Just `include_once 'hquery.php';` in your project and start using `hQuery`.
24 
25 Alternatively `composer require duzun/hquery`
26 
27 or using `npm install hquery.php`, `require_once 'node_modules/hquery.php/hquery.php';`.

30 # Usage
31 
32 ### Basic setup:
33 ```php
34 // Optionally use namespaces
35 use duzun\hQuery;
36 
37 // Either use commposer, either include this file:
38 include_once '/path/to/libs/hquery.php';
39 
40 // Set the cache path - must be a writable folder
41 // If not set, hQuery::fromURL() whould make a new request on each call
42 hQuery::$cache_path = "/path/to/cache";
43 
44 // Time to keed request data in cache, seconds
45 // A value of 0 disables cahce
46 hQuery::$cache_expires = 3600; // default one hour
47 ```

49 ### Load HTML from a file
50 ###### [hQuery::fromFile](https://duzun.github.io/hQuery.php/docs/class-hQuery.html#_fromFile)( string `$filename`, boolean `$use_include_path` = false, resource `$context` = NULL )
51 
52 ```php
53 // Local
54 $doc = hQuery::fromFile('/path/to/filesystem/doc.html');
55 
56 // Remote
57 $doc = hQuery::fromFile('https://example.com/', false, $context);
58 ```
59 
60 Where `$context` is created with [stream_context_create()](https://secure.php.net/manual/en/function.stream-context-create.php).
61 
62 For an example of using `$context` to make a HTTP request with proxy see [#26](https://github.com/duzun/hQuery.php/issues/26#issuecomment-351032382).
63

65 ### Load HTML from a string
66 ###### [hQuery::fromHTML](https://duzun.github.io/hQuery.php/docs/class-hQuery.html#_fromHTML)( string `$html`, string `$url` = NULL )
67 
68 ```php
69 $doc = hQuery::fromHTML('<html><head><title>Sample HTML Doc</title><body>Contents...</body></html>');
70 
71 // Set base_url, in case the document is loaded from local source.
72 // Note: The base_url is used to retrive absolute URLs from relative ones
73 $doc->base_url = 'http://desired-host.net/path';
74 ```

76 ### Load a remote HTML document
77 ###### [hQuery::fromUrl](https://duzun.github.io/hQuery.php/docs/class-hQuery.html#_fromURL)( string `$url`, array `$headers` = NULL, array|string `$body` = NULL, array `$options` = NULL )
78 ```php
79 use duzun\hQuery;
80 
81 // GET the document
82 $doc = hQuery::fromUrl('http://example.com/someDoc.html', ['Accept' => 'text/html,application/xhtml+xml;q=0.9,*/*;q=0.8']);
83 
84 var_dump($doc->headers); // See response headers
85 var_dump(hQuery::$last_http_result); // See response details of last request
86 
87 // with POST
88 $doc = hQuery::fromUrl(
89  'http://example.com/someDoc.html', // url
90  ['Accept' => 'text/html,application/xhtml+xml;q=0.9,*/*;q=0.8'], // headers
91  ['username' => 'Me', 'fullname' => 'Just Me'], // request body - could be a string as well
92  ['method' => 'POST', 'timeout' => 7, 'redirect' => 7, 'decode' => 'gzip'] // options
93 );
94 
95 ```
96 
97 For building advanced requests (POST, parameters etc) see [hQuery::http_wr()](https://duzun.github.io/hQuery.php/docs/class-hQuery.html#_http_wr),
98 though I recomend using a specialized library for making requests
99 and `hQuery::fromHTML($html, $url=NULL)` for processing results.
100 See [Guzzle](http://docs.guzzlephp.org/en/stable/) for eg.
101 
102 Another option is to use [stream_context_create()](https://secure.php.net/manual/en/function.stream-context-create.php)
103 to create a `$context`, then call `hQuery::fromFile($url, false, $context)`.

106 ### Processing the results
107 ###### [hQuery::find](https://duzun.github.io/hQuery.php/docs/class-hQuery.html#_find)( string `$sel`, array|string `$attr` = NULL, hQuery_Node `$ctx` = NULL )
108 ```php
109 // Find all banners (images inside anchors)
110 $banners = $doc->find('a > img:parent');
111 
112 // Extract links and images
113 $links = array();
114 $images = array();
115 $titles = array();
116 
117 // If the result of find() is not empty
118 // $banners is a collection of elements (hQuery_Element)
119 if ( $banners ) {
120 
121  // Iterate over the result
122  foreach($banners as $pos => $a) {
123  $links[$pos] = $a->attr('href'); // get absolute URL from href property
124  $titles[$pos] = trim($a->text()); // strip all HTML tags and leave just text
125 
126  // Filter the result
127  if ( !$a->hasClass('logo') ) {
128  $img = $a->find('img')[0]; // ArrayAccess
129  if ( $img ) $images[$pos] = $img->src; // short for $img->attr('src')
130  }
131  }
132 
133  // If at least one element has the class .home
134  if ( $banners->hasClass('home') ) {
135  echo 'There is .home button!', PHP_EOL;
136 
137  // ArrayAccess for elements and properties.
138  if ( $banners[0]['href'] == '/' ) {
139  echo 'And it is the first one!';
140  }
141  }
142 }
143 
144 // Read charset of the original document (internally it is converted to UTF-8)
145 $charset = $doc->charset;
146 
147 // Get the size of the document ( strlen($html) )
148 $size = $doc->size;
149 ```
